# DataBI
电信通话记录BI分析

实现功能： 用户可以查看每天，每个月的通话时间，通话次数，以及亲密度的查看。

## 技术栈：
### Springboot,mybatis,thymeleaf,layui,ajax,echarts

## 值得学习点

简单的springboot 开发，链接mysql,使用mybstis,架构规范分层，前端传参数，封装echats需要的数据。
